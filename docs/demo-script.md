# Demo Script

## 90-second Walkthrough

1. Open the dashboard and introduce it as a live command center for Mahakumbh 2028 operations.
2. Point to the four top metrics: crowd load, incidents, bus turnaround, and volunteer coverage.
3. Switch the scenario to **Royal bath surge** and show how the heatmap and briefing change.
4. Sort the incident queue by SLA to show operational prioritization.
5. Log a new incident from Gate 7 and show it entering the queue.
6. Export the command brief as a shift handover artifact.

## Judge-facing Pitch

Sangam Setu helps the event team move from reactive monitoring to proactive command. It aggregates transport, crowd, incident, volunteer, and resource signals into one screen and turns them into short, actionable recommendations for the next 15-30 minutes.

## What To Emphasize

- Practical control-room layout
- Offline-friendly static demo
- Clear future path to real-time feeds
- Human-in-the-loop AI guidance
- Built for large-scale Indian event operations, not a generic admin template
